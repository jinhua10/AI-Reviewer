# 🚀 React UI 快速启动指南

## ⚡ 3步启动

### 1️⃣ 启动后端
```bash
cd D:\Jetbrains\hackathon\AI-Reviewer\ai-reviewer-base-file-rag
mvn spring-boot:run
```

### 2️⃣ 打开浏览器
```
http://localhost:8080
```

### 3️⃣ 开始使用
- 💬 智能问答：输入问题获得 AI 回答
- 🔍 文档搜索：搜索相关文档
- 📊 统计信息：查看知识库统计

## ✨ 功能特性

### 💬 智能问答
- ✅ 输入问题，获得 AI 回答
- ✅ 显示参考来源
- ✅ 响应时间统计

### 🔍 文档搜索  
- ✅ 关键词搜索
- ✅ 可调整数量（5/10/20/50）

### 📊 统计信息
- ✅ 文档总数
- ✅ 已索引数量
- ✅ 完成度百分比

## 🎨 设计特点

- 🌈 **渐变色主题** - 紫色渐变背景
- ✨ **平滑动画** - 淡入淡出效果
- 📱 **响应式布局** - 支持手机/平板/桌面
- 🚀 **零配置** - 开箱即用

---

## 🔧 技术栈

- **React 18** - UI 框架
- **Babel Standalone** - JSX 转译
- **Axios** - HTTP 请求
- **纯 CSS3** - 样式和动画

**特点**: 单文件应用，无需构建工具！

## 🐛 常见问题

### Q: 页面打不开？
**A**: 确认后端已启动，访问 http://localhost:8080/api/qa/health 检查

### Q: API 调用失败？
**A**: 检查浏览器控制台（F12）查看错误信息

### Q: 问答没响应？
**A**: 确认已配置 LLM（API Key）和有索引的文档
