# Task Manager UI

一个使用React构建的简单任务管理前端应用，用于测试代码分析功能。

## 功能特性

- ✅ React 18+ Hooks
- ✅ 组件化设计
- ✅ 状态管理
- ✅ API集成
- ✅ 响应式UI

## 项目结构

```
src/
├── App.jsx                # 主应用组件
├── index.js              # 入口文件
└── components/
    ├── Header.jsx        # 头部组件
    └── TaskList.jsx      # 任务列表组件
```

## 组件说明

### App组件
- 主应用容器
- 管理任务状态
- 处理API调用

### Header组件
- 显示应用标题
- 提供添加任务表单

### TaskList组件
- 展示任务列表
- 支持删除操作

## 安装运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start
```
