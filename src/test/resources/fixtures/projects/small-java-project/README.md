# Simple Calculator

一个简单的Java计算器应用程序，用于测试代码分析功能。

## 功能特性

- 基本四则运算（加、减、乘、除）
- 平方和绝对值计算
- 工具方法（时间格式化、质数判断等）

## 项目结构

```
src/
├── Main.java         # 主程序入口
├── Calculator.java   # 计算器核心类
└── Utils.java        # 工具类
```

## 编译运行

```bash
# 编译
javac src/*.java

# 运行
java -cp src com.example.calculator.Main
```
