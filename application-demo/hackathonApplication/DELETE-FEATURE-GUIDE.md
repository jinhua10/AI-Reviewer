# 文件删除功能 - 快速使用指南

## 🎯 功能说明

现在用户可以在提交完成前删除已上传的 ZIP 文件。

## 📸 功能展示

### 删除按钮位置
在"已上传文件"卡片中，每个文件旁边都有一个红色的 **🗑️ 删除** 按钮（仅在未标记完成时显示）。

### 使用步骤

1. **上传文件**
   - 选择 ZIP 文件并上传
   - 文件会出现在"已上传文件"列表中

2. **删除文件**
   - 点击文件旁边的 **🗑️ 删除** 按钮
   - 系统弹出确认对话框：
     ```
     ⚠️ 确认删除
     
     确定要删除文件 "xxx.zip" 吗？
     
     此操作无法撤销。
     ```
   - 点击"确定"后，文件被删除
   - 页面刷新，文件从列表中消失

3. **限制说明**
   - ✅ 可以删除：提交完成前的任何已上传文件
   - ❌ 不能删除：标记完成后的文件
   - ❌ 不能删除：其他团队的文件

## ⚠️ 重要提示

- 删除操作**无法撤销**，请谨慎操作
- 标记提交完成后，**无法再删除文件**
- 删除按钮会在标记完成后自动隐藏

## 🔧 技术实现

### 后端接口
- **路径**: `POST /delete`
- **参数**: `fileName` (要删除的文件名)
- **返回**: 重定向到上传页面，显示成功/失败消息

### 前端交互
- 删除确认对话框防止误操作
- 删除中状态（按钮禁用，显示"删除中..."）
- 操作完成后自动刷新页面

## 🛡️ 安全保障

- ✅ 必须登录才能删除
- ✅ 只能删除自己团队的文件
- ✅ 防止路径遍历攻击
- ✅ 只能删除 .zip 文件
- ✅ 提交完成后自动禁止删除

## 📝 操作日志

所有删除操作都会记录在系统日志中：
```
INFO - 团队 T00001 删除文件: project-v1.zip
```

## 🚀 启动应用

重新启动应用后，功能即可生效：
```bash
java -jar target/hackathonApplication.jar
```

然后访问: http://localhost:8899
